    Resumo:

    Para a conseptualização do trabalho de casa, foi utilizado, como template, o formato para aplicações web criado na aula numero 5.

    De forma a resolver o desafio, de criar um painel de escolhas, entre mestrado e doutoramento, foi utilizado a opção radio, com os aspetos da stylesheet de w3.

    Quando o pathname é '/', foi criado uma pagina, utilizado uma template em pug, que contém apenas uma descrição da aplicação e dois botões, Registo e Consulta. Alem disso foi inserido em todos os footer um botão para voltar à pagina inicial.
