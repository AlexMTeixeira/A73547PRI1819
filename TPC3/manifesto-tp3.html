    Resumo:

    Para Schema de transformação do ficheiro quran.xml, foi inicialmente, criado um template para separa o Index dos ficheiros que contêm as suras.

    O ficheiro index.html foi transformado, principalmente com duas templates. A primeira com o objetivo de percorrer todas a suras e escrever o seu título, juntamente com o link para o ficheiro correspondente, num index. A seguinte, têm como o objectivo, escrever as informações do livro (title,title2,subtitle e preface).

    Para os ficheiros, suraXXX.html, foi necessário fazer referencia ao Index, com o objectivo de voltar à pagina inicial, e também escrever o conteudo de cada uma delas (bktlong, bktshort e v).

    Resultados:

    • Pasta que contêm o website
    • XML com o conteudo do manifesto.
    • Schema de conversão XML para HTML.
    • Manifesto representado em HTML.
