    Resumo:

    Inicialmente, foi necessário criar um script para transformar os ficheiros musicais ".json" num ficheiro "index.json", que contem a concatenação dos vários ficheiros das obras musicais. Para isto, foi necessário implementar o modulo fs para ler a diretoria e de seguida utilizar a função foreach() para iterar sobre a diretoria, e com isto, analisár ficheiro a ficheiro. Esta análise verifica se os ficheiros são do tipo /m[0-9+\.json/, alêm disso verifica se é o primeiro e ultimo caso, para uma correta concatenação.

    Seguidamente, foi criado um servidor, capaz de identificar, a partir do pathname, qual modulo em PUG que deve implementar. Isto é, se o pathname for igual a index, executa o render do ficheiro index.pug com o parse do ficheiro index.json. E quando, pathname é igual a obra/mXX, é executado o render do ficheiro obra.pug com o parse da obra correspondente.

    Finalmente, foi necessário alterar a porta de listen(), de 6000 para 6001, porque o servidor fica inresponsivo em todos URLs fornecidos.
