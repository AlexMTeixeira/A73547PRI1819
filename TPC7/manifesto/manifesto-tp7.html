    Resumo:

    Inicialmente foi necessário implementar um servidor, capaz de responder a dois métodos, Post e Get. Para o método Get, foi necessário verificar os pathnames, /, /ficheiros e /data/*. Para o Post foi apenas implementada uma resposta quando o pathname é igual a /processaForm.

    No link para os ficheiros ja guardados, foi necessário definir uma resposta, capaz de analisar o tipo de ficheiro e escrever o seu respetivo Header. Para isso, são utilizadas expressões regulares, juntamente com a função test para identificar corretamente o tipo de ficheiro.
